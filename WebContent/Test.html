About this project, This project is the server side for the Asset management system, Android project. This project deals with all the maintenance of the client like backup of the user data so that the user can save his data on line. This back up takes automatically on the client side periodically, and also the user can force save the data.

Initially this project is developed so that it can be used on retail shops and home users for maintaining their expenses and incomes and will show their weekly,monthly and yearly expenses on chart or any diagram format.

This project will be developed for tablets and smartphones.
